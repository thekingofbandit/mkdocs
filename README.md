# mkdocs

to get started with mkdocs.


### Structure Files

- `src` - mkdocs source code

- `site` - mkdocs build directory



### Install

- `pip install mkdocs`


### Development

- `mkdocs serve`

 Serving on http://127.0.0.1:8000  



### Building the site

- `mkdocs build`  




### Creating a new project

- `mkdocs new my-project`

- `cd my-project`
